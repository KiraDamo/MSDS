Project Overview and Data Description

Introduction

  • This project assignment focuses on data visualization using ggplot2 and employs dplyr for data manipulation.
  • You have been provided with a CSV file containing Olympic athletes’ information, including columns such as:
    • Name, Sex, Age, Height, Weight, Team, NOC, Games, Year, Season, City, Sport, Event, Medal
  • Please use only ggplot2 for creating the plots in these questions. You may use dplyr to preprocess or transform the data if needed.
  • The dataset is available in the file athlete_events.csv.

Data

The dataset contains records of Olympic athletes from various editions of the Summer and Winter Olympic Games. Each row typically represents one athlete’s participation in a specific Olympic event. This data can be used to explore relationships among athletes’ physical attributes (e.g., height, weight, age), examine how athletes are distributed among different teams and sports, and analyze the awarding of medals over time.

Explanation of Variables

ID:
A unique identifier assigned to each row in the dataset.

Name:
The athlete’s name. (Some athletes may appear multiple times if they participated in multiple events or editions of the Games.)

Sex:
The athlete’s reported sex, typically “M” for male or “F” for female.

Age:
The athlete’s age (in years) at the time of competition.
Numeric values; may include missing data coded as NA.

Height:
The athlete’s height (in centimeters).
Numeric values; may include missing data coded as NA.

Weight:
The athlete’s weight (in kilograms).
Numeric values; may include missing data coded as NA.

Team:
The name of the team the athlete represents (often the name of the country or a specific regional team).

NOC:
The National Olympic Committee code, a three-letter abbreviation representing the athlete’s country or region (e.g., “USA,” “CHN,” “GBR”).

Games:
The string representation of the exact Olympics edition, e.g., “1992 Winter,” “2000 Summer.”

Year:
The year in which the recorded Olympic Games took place.

Season:
Indicates whether the athlete participated in the Summer or Winter Olympic Games.

City:
The city in which the Olympic Games took place.

Sport:
The general sport category for the athlete’s event (e.g., “Athletics,” “Swimming,” “Gymnastics,” etc.).

Event:
The specific event within the sport in which the athlete competed (e.g., “Basketball Men’s Basketball,” “100m Freestyle Men,” etc.).

Medal:
The medal (if any) that the athlete won in their event. Typically “Gold,” “Silver,” “Bronze,” or “NA” if no medal was awarded.

2. Summer vs. Winter Participation

2.a Number of entries per season

Hint: First separate records by season, then tally.

Solution:

olympics |>
    group_by(Season) |>
    count()
# A tibble: 2 × 2
# Groups:   Season [2]
  Season      n
  <chr>   <int>
1 Summer 221167
2 Winter  48564
#Grouped the entries by season. Summer had 221167 and Winter had 48564.

2.b Visual comparison of those counts

Hint: A simple bar chart will illustrate the difference.

Solution:

olympics |>
    group_by(Season) |>
    summarize(n_season_entries = n()) |>
    ggplot(aes(x = Season, y = n_season_entries, fill = Season)) + geom_col() +
    labs(
        x = "Season",
        y = "Number of Entries",
        title = "Number of Entries in the Summer vs the Winter"
    )
