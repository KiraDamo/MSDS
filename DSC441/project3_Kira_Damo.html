Project Overview and Data Description

Introduction

  • This project assignment focuses on data visualization using ggplot2 and employs dplyr for data manipulation.
  • You have been provided with a CSV file containing Olympic athletes’ information, including columns such as:
    • Name, Sex, Age, Height, Weight, Team, NOC, Games, Year, Season, City, Sport, Event, Medal
  • Please use only ggplot2 for creating the plots in these questions. You may use dplyr to preprocess or transform the data if needed.
  • The dataset is available in the file athlete_events.csv.

Data

The dataset contains records of Olympic athletes from various editions of the Summer and Winter Olympic Games. Each row typically represents one athlete’s participation in a specific Olympic event. This data can be used to explore relationships among athletes’ physical attributes (e.g., height, weight, age), examine how athletes are distributed among different teams and sports, and analyze the awarding of medals over time.

Explanation of Variables

ID:
A unique identifier assigned to each row in the dataset.

Name:
The athlete’s name. (Some athletes may appear multiple times if they participated in multiple events or editions of the Games.)

Sex:
The athlete’s reported sex, typically “M” for male or “F” for female.

Age:
The athlete’s age (in years) at the time of competition.
Numeric values; may include missing data coded as NA.

Height:
The athlete’s height (in centimeters).
Numeric values; may include missing data coded as NA.

Weight:
The athlete’s weight (in kilograms).
Numeric values; may include missing data coded as NA.

Team:
The name of the team the athlete represents (often the name of the country or a specific regional team).

NOC:
The National Olympic Committee code, a three-letter abbreviation representing the athlete’s country or region (e.g., “USA,” “CHN,” “GBR”).

Games:
The string representation of the exact Olympics edition, e.g., “1992 Winter,” “2000 Summer.”

Year:
The year in which the recorded Olympic Games took place.

Season:
Indicates whether the athlete participated in the Summer or Winter Olympic Games.

City:
The city in which the Olympic Games took place.

Sport:
The general sport category for the athlete’s event (e.g., “Athletics,” “Swimming,” “Gymnastics,” etc.).

Event:
The specific event within the sport in which the athlete competed (e.g., “Basketball Men’s Basketball,” “100m Freestyle Men,” etc.).

Medal:
The medal (if any) that the athlete won in their event. Typically “Gold,” “Silver,” “Bronze,” or “NA” if no medal was awarded.

Questions

1. How many athletes are male vs. female?

Hint: Create a bar chart comparing the counts of male and female athletes.

Solution:

olympic_data |>
    group_by(Sex) |>
    ggplot(aes(x=Sex, fill = Sex)) + geom_bar() +
    labs(
        x = "Sex",
        y = "Count",
        title = "Number of Olympic Athletes (Female vs Male)"
        )
#Grouped by Sex and plotted as a bar plot.

2. How is age distributed among the athletes?

Hint: Use a histogram to see how ages are spread out across the dataset.

Solution:

olympic_data |>
    select(Age) |>
    na.omit() |>
    ggplot(aes(x = Age)) + geom_histogram(bins = 50, fill = "purple", color = "white") +
    labs(
        x = "Age",
        y = "Count",
        title = "Age Distribution of Olympic Athletes"
    )
#selected Age data and removed all NA values. Created a histogram with the x label as age. There are mostly athletes in their twenties but there are some athletes who are older or are in their late teenage years.

3. Is there a relationship between height and weight?

Hint: A scatter plot can show how height correlates with weight.
There are too many points in the data, so sampling the data can help visualize it better. (10,000 points is a good sample size.)

Solution:

olympic_data |>
  sample_n(10000) |>
    select(Height, Weight) |>
    ggplot(aes(x = Weight, y = Height)) + geom_point(color = "darkblue", alpha = 0.2, na.rm = TRUE) +
    labs(
        title = "Height vs Weight of Olympic Athletes",
        x = "Weight (kg)",
        y = "Height (cm)" 
    )
#Created a sample of 10,000 and assigned weight to x and height to y. Plotted scatter plot. Taller athletes tend to weigh more.

4. How do athlete weights vary by sex?

Hint: Use a box plot to compare weight distribution in different groups.

Solution:

olympic_data |>
    group_by(Sex) |>
    ggplot(aes(x = Sex, y = Weight, fill = Sex)) +
    geom_boxplot(na.rm = TRUE) +
    labs(
        title = "Weight Variations by Sex",
        x = "Sex",
        y = "Weight (kg)"
    )
#Grouped by sex and created a boxplot. Removed all NA values. Male athletes tend to weigh more than female athletes by average.

5. Which types of medals appear most frequently?

Hint: Count how often Gold, Silver, and Bronze appear (excluding NA).

olympic_data |>

6. How does athletes’ age distribution compare between Summer and Winter Olympics?

Hint: Facet by Season (Summer vs. Winter) to visually compare two histograms.

Solution:

olympic_data |>
    group_by(Season, na.rm =TRUE) |>
    ggplot(aes(x = Age, fill = Season)) +
    geom_histogram(bins = 30, position = "stack", alpha = 0.5, na.rm = TRUE) +
    labs(
        title = "Age distribution by Season (Summer vs. Winter)",
        x = "Age",
        y = "Count",
        fill = "Season"
    )
#Grouped by Season and removed all NA values. Created a histogram that was "stacked" and had an alpha of 0.5.
#Summer athletes and winter athletes tend to be around the same age on average.

7. How are medals distributed across different sports?

Hint: Group by “Sport,” count medals, then create a horizontal bar chart (exclude athletes with no medal).
Reorder the sports by the count of medals for better visualization.

Solution:

olympic_data |>
    group_by(Sport) |>
    filter(Medal %in% c("Bronze", "Silver", "Gold"), na.rm = TRUE) |>
    summarize(n_medals = n()) |>
    ggplot(aes(x = reorder(Sport, n_medals), y = n_medals)) + geom_col(fill="darkgreen") + coord_flip() +
    labs(
        title = "Medal Distribution Across Different Sports",
        y = "Number of Medals",
        x = "Sport"
    )
#Grouped by sport and counted only athletes who had a medal. Counted the number of medals that were earned in each sport and plotted a bar chart.

8. Do medalists differ in height from non-medalists?

Hint: Create a new variable that categorizes athletes as medalists or non-medalists, then compare their heights using a box plot.

Solution:

olympic_data |>
    mutate(medalists = ifelse(Medal %in% c("Bronze", "Silver", "Gold"), "Medalist", "Non-Medalist")) |>
    ggplot(aes(x = medalists, y = Height, fill = medalists)) + geom_boxplot(na.rm = TRUE) +
    labs(
        x = "Medalists",
        y = "Height (cm)",
        title = "Heights of Medalists vs Non-Medalists"
    )
#Created a new column that identifies if athlete earned a medal or not. Plotted a boxplot and filled according to medalist or not.
#They are almost the same but medalists' heights are slightly taller than non-medalists.

10. Which teams have the largest number of athletes?

Hint: Select the top 10 teams after counting athletes per team, then plot in descending order.

Solution:

olympic_data |>
    group_by(Team) |>
    summarize(n_team_members = n()) |>
    arrange(desc(n_team_members)) |>
    head(n=10) |>
    ggplot(aes(x = n_team_members, y =reorder(Team, n_team_members))) + geom_col(fill="darkgreen") +
    labs(
        title = "Teams With The Most Athletes",
        x = "Number of Athletes",
        y = "Team"
    )
#Grouped by team, counted team members for each team. Sorted the Teams in decreasing order and returned the top 10 Teams. Plotted a bar chart in decreasing order.

#The largest teams are the United States, France, Great Britain, Italy, Germany, Canada, Japan, Sweden, Australia, and Hungary.
